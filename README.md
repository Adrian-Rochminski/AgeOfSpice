#Age of Spice

Age of Spice to turowa gra strategiczna dla 3 graczy grających na jednym komputerze, napisana z wykorzystaniem technologii JavaFX.
Powstała w ramach zaliczenie przedmiotu Komunikacja Człowiek-Komputer.
